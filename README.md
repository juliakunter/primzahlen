# Primzahlen  
**Author:** Kunter  Julia

Berechnung von Primzahlen
Darstellung einer logarithmischen Funktion mit Sternen

Idee
Die Seite soll zeigen, wie einfache mathematische Berechnungen im Web umgesetzt werden können.
Dabei wird die Verbindung von HTML (für die Eingabe und Ausgabe) und PHP (für die Logik) genutzt.

Aufgabenbeschreibung
1. Primzahlenberechnung

Auf einer HTML-Seite sollen alle Primzahlen von 1 bis zur eingegebenen Zahl ausgegeben werden.
Tipp: Jede Zahl durch alle Zahlen von 2 bis n/2 dividieren, um zu prüfen, ob sie eine Primzahl ist.

2. Logarithmische Funktion

Berechne den log(n) für alle Zahlen von 1 bis 100.
Gib für jedes Ergebnis eine Zeile mit Sternen aus, wobei die Anzahl der Sterne proportional zum Logarithmuswert ist.
